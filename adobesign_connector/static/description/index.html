Adobe sign | Odoo Adobe Sign Integration | Odoo Adobe Sign Connector | AdobeSign

CFIS

This module allows the odoo users Integration with Adobe Sign. We will send emails to users asking them to sign the document (Invoice and Sale Order) after setting up the Adobe Sign email. Use the Adobe Sign to retrieve the document.

Features

- Send Documents (Invoice / Sale Order) to Partners for Sign using adobesign

- List view and Form View for adobesign Sent documents

Installation

- Copy adobesign_connector module to addons folder

- Install the module normally like other modules.

1. adobesign Configure Apps and Keys

https://secure.in1.adobesign.com/public/login#pageId::ACCOUNT_SETTINGS

Add new Apps and Integration Keys


Configure OAuth for Application: Set Scope and Redirect URI (eg. https://odoo.cfis.com/adobesign/callback).


2. Odoo Configure for adobesign

Add Region, Redirect URI(HTTPS), Client Id and Client Secret


3. Odoo User Settings

Settings -> Users & Companies -> Users -> User -> adobesign


4. Send Sale Order / Invoice PDF report to adobesign

Sales -> Quotations -> Get Signature
